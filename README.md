# 微博社交机器人“评论罗伯特”的人机互动分析——基于开源工具的数据可视化与内容分析作品

一、项目简介
本项目以新浪微博社交机器人 “评论罗伯特” 为研究对象，选取微博账号 “罗伯特受害者联盟” 发布的相关博文作为数据样本，通过定量统计分析与人工编码相结合的方法，探究社交机器人在微博平台中的人机互动特征及其引发的用户情绪反应结构。

项目以数据可视化作品（Dashboard）为主要呈现形式，重点分析整体互动规模、时间分布特征，以及高点赞（万赞）博文中“评论罗伯特”的互动类型与用户情绪反应模式，旨在从数据视角呈现当代社交平台中人机互动的结构性特征。

本项目同时作为课程期末大作业，并参与 OpenSODA 2025 数据作品类（W 类）比赛。


二、基于的开源项目与技术框架说明（比赛要求）

本项目在研究设计与技术实现过程中，基于 OpenSODA 2025 提供的开源数据分析思路与工具框架，并参考 GitHub 上的相关开源代码进行数据采集与处理，在此基础上完成二次分析与可视化展示。

OpenSODA 2025 开源项目参考：
https://atomgit.com/x-lab/OpenSODA2025

数据处理与分析阶段参考并使用了 GitHub 开源代码 https://github.com/dataabc/weiboSpider（Python 爬虫与数据清洗相关模块），最终结合 DataEase 完成数据可视化 Dashboard 的构建。


三、研究对象与数据来源
   1. 研究对象
   微博账号：“罗伯特受害者联盟”
   研究内容：与微博机器人“评论罗伯特”相关的互动博文
   
   2. 数据范围
   时间范围：2023 年 12 月 21 日 — 2025 年 3 月 23 日
   原始抓取博文编号：1 — 7725
   清洗后有效样本量：6736 条博文
   
   3. 数据字段
   包括但不限于：
   博文正文
   发布时间
   点赞数 / 评论数 / 转发数
   评论罗伯特的回复内容
   用户互动反馈
   
   此外，研究进一步选取 点赞数 ≥ 10000 的博文 作为高共鸣样本，进行人工编码与内容分析。

四、研究方法与技术路线
   1. 研究方法
   定量统计分析
   
   人工编码的内容分析法
   
   数据可视化分析（Dashboard）

   2. 技术路线
      微博数据采集（Python 爬虫）
           ↓
     数据清洗与结构化处理
             ↓
     整体互动指标统计分析
             ↓
     万赞博文人工编码（互动类型 / 情绪反应）
             ↓
     DataEase Dashboard 可视化呈现
             ↓
     结果分析与研究总结

五、数据可视化作品（Dashboard）说明

本项目构建了两个核心 Dashboard 面板：

   面板一：整体互动结构分析

         博文总数指标

         覆盖时间范围
         
         平均点赞数
         
         博文数量时间分布（折线图）
         
         点赞数整体结构（柱状图）

   面板二：万赞博文互动特征分析

         万赞博文数量指标

         互动类型 × 用户情绪反应（热力图）
         
         涉及话题词云
         
         互动类型分布（饼图）
         
         用户情绪反应结构（柱状图）

Dashboard 以静态可视化成果形式展示，重点服务于数据解读与研究分析。
