# Code说明

本项目的数据采集与预处理阶段基于 GitHub 上的开源代码完成，未对原有算法逻辑进行修改。

## 代码来源

本研究参考并使用以下开源项目进行微博数据抓取与初步处理：

- 项目名称：weiboSpider
- 开源仓库：https://github.com/dataabc/weiboSpider
  
该项目主要用于：
- 微博数据的爬取
- 基础字段提取（博文正文、发布时间、互动量等）
- 原始数据结构化输出

## 本项目中的使用方式

在本研究中，开源代码被用于数据获取与初步清洗阶段，研究者在此基础上：

1. 对原始数据进行筛选与清洗，剔除缺漏数据与非研究对象内容；
2. 构建符合研究需求的数据表结构；
3. 将处理后的数据导入 DataEase 平台，完成后续统计分析与可视化展示；
4. 在高点赞样本中引入人工编码方法进行内容分析。

## 说明

本项目不涉及对原有开源算法的二次开发，代码的作用主要在于保障数据获取与处理流程的可复现性。
